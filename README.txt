Inhalt des Archivs
------------------
In diesem Archiv finden Sie die OSCI-Bibliothek in der 
Version 2.0.1. Die vorliegende OSCI-Bibliothek setzt die 
OSCI-Transport 1.2-Spezifikation um. Die OSCI-Spezifikation wird 
durch die Koordinierungsstelle für IT-Standards (KoSIT) 
herausgegeben. Die Spezifikationsdokumente finden Sie auf den 
Seiten der KoSIT unter 
https://www.xoev.de/downloads-2316#Standards 



Enthalten ist folgendes:
- die .NET-Version der OSCI-Bibliothek für .NET 3.5 (/lib3.5)
- die .NET-Version der OSCI-Bibliothek für .NET 4.0 (/lib4.0)
- die Quellen der OSCI-Bibliothek (/src)
- die API-Dokumentation (/apidoc)
- die Versionshistorie (OSCI-Bibliothek_Versionshistory.pdf).

Die vorliegende OSCI-Bibliothek ist für den Gebrauch im 
Microsoft® .NET-Kontext implementiert. Sie finden alle benötigten
DLLs im lib-Ordner. Dieser Ordner beinhaltet neben der 
eigentlichen OSCI-Bibliothek (.NET) zusätzliche Windows-
Bibliotheken (DLL-Dateien), die für den Betrieb der 
OSCI-Bibliothek notwendig sind. Aktuell ist dies die 
Bouncy-Castle-Bibliothek:
https://www.bouncycastle.org/csharp/index.html
